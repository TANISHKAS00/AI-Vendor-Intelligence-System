# AI-Powered Vendor Intelligence and Fraud Detection System

## Overview

This project is an AI-based Vendor Intelligence System developed to analyze vendor invoice data and identify business risks. The system helps organizations monitor vendor performance, detect suspicious transactions, identify duplicate invoices, and generate procurement insights.

## Features

* Fraud Detection using Isolation Forest
* Vendor Risk Scoring
* Vendor Performance Ranking
* Duplicate Invoice Detection
* Spend Trend Analysis
* Interactive Streamlit Dashboard

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Streamlit

## Dataset

The project uses vendor invoice and procurement data containing:

* Vendor Information
* Invoice Details
* Purchase Orders
* Payment Records
* Freight Costs
* Transaction Amounts

## Project Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Feature Engineering
4. Fraud Detection
5. Vendor Risk Analysis
6. Performance Ranking
7. Dashboard Development
8. Deployment

## Installation

```bash
pip install -r requirements.txt
```

## Run Application

```bash
streamlit run app.py
```

## Results

The system successfully identifies suspicious invoices, evaluates vendor risk, ranks vendor performance, and provides actionable insights through an interactive dashboard.
